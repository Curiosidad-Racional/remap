* remap

- Command:

#+begin_src sh
remap [-s] [-v]
#+end_src

- -s :: swap /Control/ key and /AltGr/ key (right Alt)
- -v :: verbose log output

- Helper command:

#+begin_src sh
sudo ./run.sh [-s] [-v]
#+end_src

write ~sudo ./run.sh~ command, with yout preferred options, press /Enter/ and wait until prompt shows again.
